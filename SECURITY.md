# Política de Seguridad

Nos tomamos en serio la seguridad de KBeca. Este repositorio es seguro para ser clonado o "forkeado".

## ⚠️ Advertencias para Contribuyentes

- **No uses la base de datos de producción** en entornos locales.
- **Usa variables de entorno** para cualquier secreto y asegúrate de que `.env.local` no se suba al repositorio.
- **Cuidado con los scripts destructivos**. Scripts como `clear-db.ts` eliminan todos los datos.

## Reporte de Vulnerabilidades

Por favor, no abras un *Issue* público si encuentras una vulnerabilidad.
Si tienes un reporte de seguridad, por favor escríbenos directamente a través de un mensaje privado al mantenedor o a la cuenta oficial del proyecto.
